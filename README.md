\# Fountain Screenplay Online

\### by Vinimay Kaul



\# What does this do?

This application is just a web view for writing and viewing Screenplays written in fountain markup. You can also print it or Print it to PDF. 





\## Dependency

This application requires fountain.min.js from \[Matt Daly's GitHub repository](https://github.com/mattdaly/Fountain.js). Please download and place it in the same directory as the other files.
